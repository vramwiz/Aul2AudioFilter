# Aul2AudioFilter

AviUtl ExEdit2 用の音声フィルタープラグインです。

プラグイン上の表示名は「サウンドエフェクター」です。声や音声素材に、電話風、無線風、反響、こもり、ナレーション補正などの効果をかけられます。

## 現在の機能

- `プリセット`: エコー、ナレーションなどの初期設定を日本語で選択
- `Delay`: 単発ディレイ、エコー、Ping-Pong ディレイ
- `EQ`: 低域カット、高域カット、帯域制限
- `Compressor`: 音量差の補正
- `Limiter`: 音割れ防止のピーク抑制
- `Distortion`: 歪みや拡声器風の質感
- `Noise`: ホワイトノイズ、クラックルノイズ
- `BitCrusher`: 低音質通話や古い機械音声のような粗い質感
- `Chorus`: 揺れのあるコーラス効果
- `Reverb`: 部屋や空間の残響

## 配布ファイルの配置

使うときは、AviUtl2 のプラグインフォルダに `Aul2AudioFilter.auf2` をコピーします。

```text
C:\ProgramData\aviutl2\Plugin\Aul2AudioFilter\Aul2AudioFilter.auf2
```

配置先フォルダ:

```text
C:\ProgramData\aviutl2\Plugin\Aul2AudioFilter
```

コピーするファイル:

```text
Aul2AudioFilter.auf2
```

## グループへの適用

複数の音声素材へまとめて効果をかけたい場合は、AviUtl2 の「グループ制御（音声）」を使います。
通常の「グループ制御」ではなく、「グループ制御（音声）」にサウンドエフェクターを追加してください。

## プリセット

- `プリセット`: 用途別の初期設定を選択
- `プリセット適用`: 選択したプリセットを詳細パラメータへ反映

`プリセット` はサウンドエフェクターの一番上に表示される日本語の選択項目です。
現在は `なし`、`エコー`、`反響`、`ホール`、`空間`、`ナレーション`、`電話`、`無線`、`拡声器`、`劣化` を選択できます。
選択後に `プリセット適用` を押すと、詳細エフェクトのパラメータへ反映されます。
反映後は各詳細パラメータを手動で微調整できます。
`なし` を適用すると、プリセット管理対象の詳細エフェクトを既定値へ戻して OFF にします。

プリセットは音声を直接処理する固定スタイルではなく、詳細エフェクトの初期値をまとめて入れるための機能です。
そのため、プリセット適用後に `Delay`、`EQ`、`Compressor` などを調整して、素材に合わせた音へ仕上げられます。

| プリセット | 反映される主なエフェクト | 用途 |
| --- | --- | --- |
| `エコー` | `Delay` | 同じ音を遅らせて繰り返す |
| `反響` | `Delay` | 左右へ跳ね返るようなエコー |
| `ホール` | `Reverb` | 広い場所の残響 |
| `空間` | `Chorus` | 左右に広がる音 |
| `ナレーション` | `EQ`, `Compressor`, `Limiter` | 声を聞きやすく整える |
| `電話` | `EQ`, `Distortion`, `BitCrusher` | 電話越しのような音 |
| `無線` | `EQ`, `Distortion`, `Noise`, `BitCrusher`, `Limiter` | 無線機のような荒い音 |
| `拡声器` | `EQ`, `Compressor`, `Distortion`, `Limiter` | メガホンや場内放送のような音 |
| `劣化` | `EQ`, `Noise`, `BitCrusher` | 古い機械音声や低音質な音 |

エコーや残響は、音声オブジェクトの長さを超えて鳴り続けることはできません。
必要な場合は、音声素材の後ろに無音部分を足してから適用してください。
